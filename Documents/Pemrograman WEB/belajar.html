Belajar HTML

HTML adalah bahasa markup yang digunakan untuk membuat halaman web.

HTML terdiri dari berbagai elemen yang digunakan untuk menyusun konten di halaman web.

Beberapa elemen HTML yang umum digunakan antara lain:

  • Heading: Digunakan untuk membuat judul atau subjudul. Contoh: <h1>, <h2>, dll.
  • Paragraph: Digunakan untuk membuat paragraf teks. Contoh: <p>.
  • Link: Digunakan untuk membuat tautan ke halaman lain. Contoh: <a href="url">.
  • Image: Digunakan untuk menyisipkan gambar. Contoh: <img src="gambar.jpg" alt="Deskripsi gambar">.

HTML juga mendukung penggunaan atribut untuk memberikan informasi tambahan pada elemen. Contoh atribut yang umum digunakan adalah:

  • class: Digunakan untuk memberikan kelas pada elemen. Contoh: <div class="container">.
  • id: Digunakan untuk memberikan identifikasi unik pada elemen. Contoh: <h1 id="judul">.
  • style: Digunakan untuk memberikan gaya langsung pada elemen. Contoh: <p style="color: red;">.

Dengan menggunakan HTML, kita dapat membuat halaman web yang menarik dan informatif.
